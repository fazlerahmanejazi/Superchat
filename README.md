# Superchat

### a simple API-service with functionalities
  * Create contacts given their personal information(name, e-mail, etc)
  * List all contacts
  * List all previous conversations

### used following to achieve this
  * Java 11
  * Spring Boot 2.5.2
  * Maven
  * PostgreSQL
  * Hibernate
